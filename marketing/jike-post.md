**文本：**

AI 生成的 HTML，换个设备就废了。微信里打不开，发给别人只能截图。

所以做了个小工具：一个 API call 进去，一个公网 URL 出来。

给 OpenClaw 做了个 skill，对 agent 说 "publish this" 就完事了。

零注册，零配置，agent 装了就能用。

clawhub install shippage
shippage.ai
